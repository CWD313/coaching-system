Coaching System - Multi-tenant SaaS (Scaffold)

This repository contains a scaffold for a coaching-management SaaS with:
- Backend: Node.js + Express + MongoDB
- Frontend: Next.js + TailwindCSS
- Auth: JWT
- Storage: AWS S3 (utility included)
- Payment: Razorpay webhook handling (basic)

See `server/README.md` and `client/README.md` for details.
